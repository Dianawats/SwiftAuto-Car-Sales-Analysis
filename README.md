# Car Sales Project
This project analyzes car sales and profits for SwiftAuto Traders.

Part 1 (Excel)
- Bar chart: Quantity Sold by Dealer ID
- Line chart: Profit by Date and Model
- Column chart: Profit by Year and Dealer ID
- Line chart: Hudson model profits by Dealer ID

Part 2 (IBM Cognos Analytics)
- KPI metrics: Profit, Quantity Sold, Average Quantity
- Bar chart: Quantity Sold by Model
- Column chart: Profit by Dealer ID
- Recalls by model
- Customer sentiment treemap
- Monthly sales vs profit
- Heatmap of recalls by system

## Tools Used
- Microsoft Excel
- IBM Cognos Analytics
